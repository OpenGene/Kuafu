# Kuafu: data analytics and machine learning library with a focus on deep learning

![Kuafu](logo/kuafu.jpg)


### Features
- State-of-art or well-known machine learning algorithms
- Deep Learning
- Data analytics with R
- Friendly APIs
- Rich visualization
- GPU acceleration


### Examples

	knn = KNearestNeighbor()
